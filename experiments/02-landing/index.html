What it does

Review diffs the way they were written — one commit at a time.

No new workflow for authors. No configuration. Open a PR, click DiffCast.

⏱
Temporal replay
Drag a slider to scrub through commits in order. Watch lines appear in the sequence they were written — not as a flattened blob.
Core feature
🔴
Ambient risk coloring
Lines glow green, amber, or red based on a risk score: lines changed + cyclomatic complexity delta + file type weight + author churn history.
High-risk detection
⚠
Dangerous window detection
Flags the window between commits where auth was added but the role guard hadn't landed yet. The most common source of security gaps in review.

⊟
Smart split pane
When a commit touches multiple high-risk files, DiffCast automatically opens a 60/40 split view so you see both simultaneously.
